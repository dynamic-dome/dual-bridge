## Ziel
Lege `overnight_demo/roman.py` an (Wegwerf-Testziel) mit einer kleinen Umwandlung ganzer Zahlen in römische Ziffern.

## Done-Kriterien
- [ ] Eine Funktion `to_roman(n: int) -> str` existiert in `overnight_demo/roman.py` und unterstützt mindestens den Bereich 1..3999.
- [ ] Beispiele stimmen: to_roman(4) == "IV", to_roman(9) == "IX", to_roman(58) == "LVIII", to_roman(1994) == "MCMXCIV".
- [ ] Bei ungültiger Eingabe (n < 1 oder n > 3999 oder kein int) wird ein `ValueError` mit aussagekräftiger Meldung geworfen.
- [ ] Die Funktion hat einen Docstring.
