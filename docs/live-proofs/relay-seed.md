## Ziel
Eine kleine Python-stdlib-Bibliothek für einfache Textstatistiken (z.B. Wortzahl, Zeichenzahl, häufigste Wörter). Erweitere sie Schritt für Schritt um sinnvolle, zusammenhängende Funktionen.

## Leitplanken
- nur Python-Standardbibliothek, keine externen Dependencies
- jeder Schritt bringt mindestens einen kurzen Test mit (assert genügt)
- kleine, in sich abgeschlossene Schritte
